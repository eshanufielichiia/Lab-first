🌐 WEB-ТЕХНОЛОГІЇ

Глосарій основних термінів

HTML
HyperText Markup Language — мова розмітки гіпертексту. Використовується для створення структури веб-сторінок.
CSS
Cascading Style Sheets — каскадні таблиці стилів. Відповідають за візуальне оформлення та адаптивність сайтів.
JavaScript
Мова програмування для створення клієнтських сценаріїв. Додає динаміку та інтерактивність на веб-сторінки.
