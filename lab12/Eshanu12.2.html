Git Command Line Reference

Повний довідник команд Git для ефективної роботи

Create a Repository
$ git init [project name]
Створити новий локальний репозиторій
$ git clone my_url
Скачати існуючий репозиторій
Show Changes (Перегляд змін)
$ git status
Список нових або змінених файлів
$ git diff
Зміни в файлах, які ще не в stage
$ git diff --cached
Зміни в staged файлах
$ git diff HEAD
Всі зміни (staged + unstaged)
$ git diff commit1 commit2
Різниця між двома комітами
$ git blame [file]
Хто і коли змінював файл
$ git show [commit]:[file]
Зміни для коміту/файлу
$ git log
Повна історія змін
$ git log -p [file/directory]
Історія змін з деталями
Working with Branches (Робота з гілками)
$ git branch
Список локальних гілок
$ git branch -a
Всі гілки (локальні + віддалені)
$ git checkout my_branch
Перейти на гілку
$ git branch new_branch
Створити нову гілку
$ git branch -d my_branch
Видалити гілку
$ git checkout branch_b
$ git merge branch_a
Злити гілку branch_a у branch_b
$ git tag my_tag
Позначити коміт тегом
Make a Change (Внесення змін)
$ git add [file]
Додати файл до stage
$ git add .
Додати всі змінені файли до stage
$ git commit -m "commit message"
Зробити коміт
$ git reset [file]
Прибрати файл з stage (зміни залишаються)
$ git reset --hard
Повернутися до останнього коміту
Synchronize (Синхронізація)
$ git fetch
Отримати зміни з origin (без злиття)
$ git pull
Fetch + merge
$ git pull --rebase
Fetch + rebase
$ git push
Відправити зміни на сервер
Finally! (Довідка)
$ git command --help
When in doubt, use git help
